# Performance Marketing Landing Page

A modern, responsive landing page built with **Vite**, **SCSS**, and **vanilla JavaScript**.  
Includes animated sections, mobile navigation, contact form, and smooth‑scroll interactions.

---

## 🚀 Tech Stack

- **Vite** — fast dev server & bundler  
- **SCSS** — modular styles with variables & mixins  
- **JavaScript (ES Modules)** — interactions & UI logic  
- **BEM‑based architecture** — clean and scalable structure
